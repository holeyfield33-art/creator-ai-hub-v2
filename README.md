# AI Starter Template

This repo is a reusable template to make Copilot work consistently across projects.

## How to use
1) Copy `ai/` into any repo
2) In Copilot Chat, start with:
   "Read ai/rules.md and follow it. Then follow ai/workflows.md."

## Default commands
- `npm test`
- `npm run typecheck`
- `npm run lint`
