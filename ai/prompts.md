# Copy/Paste Prompts (Use in Copilot)

## Fix a bug (smallest change only)
You are a senior engineer. Fix the described bug with the smallest possible change.
Rules:
- Do not refactor unrelated code.
- Do not add features.
- Add/adjust minimal tests or a validation script.
Output:
1) Files changed
2) Patch (code)
3) Commands to run
4) How to verify

## Implement an endpoint (strict)
Implement endpoint exactly as specified. No extras.
Include:
- route handler
- validation
- error handling
- tests
- docs update
